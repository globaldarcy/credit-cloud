# Vue 响应式页面脚手架

## 安装项目依赖包
```bash
npm install
```

## 运行项目
```bash
npm start
```

## 打包编译项目

输出到根目录 `dist` 文件夹
```bash
npm run build
```

## 项目简单说明

`common` 文件夹下放公共元素, `common` 文件夹里的 `imgs` 文件夹, 用来存放写在 `css` 里的图片. 小图片会自动转换成 `base64` , 提供加载速度.

`component` 存放 `header` 和 `footer` 组件.

`pages` 所以页面都放在里面, 目前层级只支持两级.

`static` 是用来存放静态资源.
